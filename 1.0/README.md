# 🥥 COCONUT WARS 椰子戰爭 · 後台系統

依《工作人員說明書（COW）》規則製作的網頁版遊戲主持後台。海島風情、可愛卡通島嶼風格。

## 快速開始

因使用 ES Modules，需透過本機伺服器開啟（不能直接雙擊 `index.html`）：

```bash
cd cscampearthgame
python3 -m http.server 8000
# 瀏覽器開啟 http://localhost:8000
```

### 雙視窗（主控台 + 播放視窗）

- 開啟 `http://localhost:8000` 即為**主控台**（工作人員操作：設定、下指令、結算）。
- 主控台右上角按 **「🖥️ 開啟播放視窗」**，會另開一個**播放視窗**（`?view=display`）給小隊員觀看戰況：只顯示大地圖、兵營、排行榜與結算動畫，沒有任何操作按鈕。
- 兩個視窗透過瀏覽器 `BroadcastChannel` 即時同步；主控台一按結算，播放視窗會同步播放動畫並更新盤面。
- 需為**同一瀏覽器、同一網址來源**（同一台電腦的多個視窗/分頁）。可把播放視窗拖到投影幕全螢幕。

## 遊戲流程

1. **基本設定**：輸入小隊數量（預設 10）與回合數量（預設 4）。小隊編號為 **0 ~ 小隊數-1**（10 隊 = 第 0 隊 ~ 第 9 隊）。
2. **島嶼歸屬**：指派 5 大島、7 小島的初始佔領隊伍；金錢島 / 軍火島固定第二回合開放、起始中立。
3. **各隊總兵力**：輸入每隊累積總兵力（大島擁有者島上自動 +1000、小島 +500，其餘進兵營）。
4. **盤面**：地圖上每座島嶼**持續顯示佔領小隊與兵力**；地圖**下方沙灘上有 10 頂兵營帳篷**（0~9 隊），顯示各隊兵營兵力。
5. **回合操作**：每隊每回合最多 3 個指令。點四種模板按鈕（`move` 運兵 / `help` 協防 / `accept` 開城 / `attack` 進攻）後，畫面**同時顯示兩種輸入方式**，任選其一：
   - **🖱️ 用選的**：下拉選單 / 勾選夥伴。
   - **⌨️ 用打的**：直接打字（逗號或空白分隔），例如 `attack`：`兵營, 金錢島, 500, 1 2`（夥伴可留空）。
   - 展開「📖 可用名稱參考」可查島嶼名稱與此隊可用出發地。
6. **結算動畫**：按「結算本回合」後，依階段逐一播放動畫並顯示當前階段名稱：
   - 🚶 移動階段（小人移動）→ 🚩 結盟階段（插旗）→ 🛡️ 協防階段（盾牌進駐）→ ⚔️ 攻擊階段（出兵＋開戰爆擊）→ 🥥 資源結算（椰子飄出）
   - 動畫同時在畫面左下角顯示 **「📢 結算實況」跑馬燈**，逐階段列出事件文字（結盟、守方兵力、攻下/守住結果、遣返、利息、救濟…），**主控台與播放視窗都會同步顯示**。
   - 動畫播放時盤面維持結算前狀態，結束後才更新為新盤面並列出文字結算紀錄。
7. **最終排名**：所有回合結束後依椰子總數排名。

## 已實作的規則重點

- 進攻/協防最低 300 兵、100 為單位；出發地兵力超支 → 相關指令全數遣返
- 第一回合資源點封鎖（畫面反黑）
- `help` / `accept` 全員互相指名配對，不成立即遣返
- `attack` 合法集合（互相指名 clique）+ 四階衝突裁決（國家數 → 出兵數 → 全國國力 → 隨機）
- 開戰比大小：進攻破城（存活兵依出兵比例留駐）／防守成功（50% 保底 + 20% 戰俘收編）／同歸於盡與漁翁得利
- 資源依駐軍比例分配；軍火島兵力注入兵營
- 第一回合結束兵營 ×1.5；每回合末低於 1000 兵救濟補滿

## 檔案結構

```
cscampearthgame/
├── index.html            入口頁
├── assets/
│   ├── css/style.css     海島風情樣式
│   └── js/
│       ├── config.js     島嶼定義、地圖佈局、配色、規則常數
│       ├── state.js      遊戲狀態模型與查詢
│       ├── engine.js     結算引擎（結盟裁決、開戰、資源）
│       ├── ui.js         介面渲染（設定 / 地圖 / 指令 / 結算 / 排名）
│       └── main.js       應用進入點與回合流程
├── docs/rules.md         原始規則說明書
└── island/               島嶼圖片（金錢島=金幣島、軍火島=齒輪島）
```
