# 🥥 COCONUT WARS 3.0 · 完整規則書

> 目標一樣：**回合結束時，椰子（分數）最多的小隊獲勝。**
> 相對 2.0：**移除「開墾」**，玩家指令改為 **移動 / 攻擊** 兩種；攻擊平手改為**各折損 500**；養兵的「訓練」改為**每回合自動生效**；並新增**島嶼守軍下限（不足 100 強制中立）**。

---

## 一、名詞

| 名詞 | 說明 |
| :--- | :--- |
| **小隊** | 玩家隊伍，編號 **0 ~（小隊數−1）**（10 隊 = 第 0 小 ~ 第 9 小）。 |
| **兵力** | 行動資源，如同「錢」。以 **100 為最小單位**。 |
| **椰子** | 分數，只會累加、不會被扣，最後比誰多。 |
| **島嶼** | 據點，共 **9 座**：四周 **8 座小島**（各隊領地）＋中央 **1 座大島「黃金洞窟」**；黃金洞窟為開局中立寶島（無守軍的空島，先搶先得、可被攻佔）。 |
| **專屬兵營** | 每隊的家，兵營內的兵是機動兵力，**絕對安全、不可被攻擊**。 |
| **駐軍** | 留在某島上的兵力，即該島的防守力量。 |

---

## 二、賽前設定

1. 輸入小隊數（預設 10）與回合數（預設 4）。
2. 島嶼歸屬：設定每座島開局由哪隊佔領，或設中立（中央的黃金洞窟通常留中立）。
3. 各隊累積總兵力（沿用前置大地遊戲結果）。
4. 初始分發（自動）：**總兵力全數放入兵營**；每座己方島嶼**額外 +500 駐軍**（大島、小島皆 500），這 500 是**額外贈送、不從總兵力扣除**。
   - 例：某隊總兵力 4000、開局持有 3 座島 → 兵營 4000 ＋ 三島各 500 = 全場 5500 兵。

---

## 三、回合流程

- 共進行設定的回合數（預設 4）。
- **每隊每回合最多 3 個指令**（移動 / 攻擊，可重複類型）。**訓練不需下指令、每回合自動生效**（見下）。
- 提交後，主控台結算，系統**依固定順序**處理：

> **結算順序：① 移動 → ② 攻擊 → ③ 自動訓練 → ④ 資源發放（結算椰子）**

各階段**依序執行**：先做完所有移動，再處理所有攻擊，接著自動訓練，最後發椰子。因此**可以先移動、再從新位置攻擊**（移動先於攻擊）。資源發放前會先套用「守軍下限」檢查（見第四節）。

---

## 四、兩種指令動作

出發地一律是**己方的兵營或己方島嶼**；兵力一律為 **100 的倍數**。

### 1. 🚶 移動 `移動(出發, 目的, 兵力)`
- 在**自己的兵營與自己的島嶼之間**調度兵力。
- **每個移動指令最多 500 兵** → 一回合 3 個移動指令最多搬 **1500 兵**。
- **第一回合移動不限額**（單指令可搬任意數量，仍為 100 的倍數、不可超過來源兵力）；第二回合起才有 500 上限。
- 目的地必須是己方兵營或己方島嶼。

### 2. ⚔️ 攻擊 `攻擊(出發, 目標, 兵力)`
- 從己方領地派兵攻擊一座**不是自己的**島。
- **無最低門檻**（100 的倍數即可），單一指令無上限。
- **同一隊多來源打同一島 → 先全部加總**，當作該隊本次的攻擊兵力。
- 詳細裁決見第五節。

> 玩家能主動下的指令只有「移動」與「攻擊」兩種。以下的「自動訓練」與「守軍下限」是每回合系統自動處理的機制，不需下指令。

### 🏋️ 自動訓練（每回合自動，全隊皆同）
- 每回合結算時，對每隊的**兵營**自動練兵：**兵營每滿 1000 兵，+200 兵**（即 `floor(兵營 ÷ 1000) × 200`，加進兵營）。
  - 例：兵營 5000 → +1000（5×200）；兵營 5900 → +1000（仍是 5 個整千）。
- **不需下指令、也不能下指令**：全隊每回合自動生效一次。
- 訓練在**移動、攻擊之後**結算，故計算基準是**當回合移動、攻擊處理完後的兵營兵力**（攻擊失敗／退回兵營的兵，會算進來；已在島上的兵不算）。

### 🛡️ 守軍下限（不足 100 強制中立）
- 己方每座島**至少須留 100 駐軍**。每回合結算（含攻防、退兵、訓練）後，若某座**己方島嶼守軍不足 100**，該島**強制變回中立空島**（棄守，殘餘駐軍消失），該回合起不再為原主產出椰子。
- 由於所有兵力皆為 100 的倍數，實務上等同「把島守到 0（整批搬空）就會失去該島」；此規則明文化，避免以極少兵力空守。

---

## 五、攻擊裁決

先把每隊「對同一座島」的所有攻擊兵力**加總**成一支部隊，再對**每座被攻擊的島**裁決。分兩階段。

### 第一階段：攻方之間選出「突圍者」
反覆進行：
1. 比較目前所有攻方中**兵力最高者**。
2. **若唯一最高** → 該隊即為**突圍者**；其餘攻方**把兵拿回原本的出發地、喪失攻擊資格**（保留兵力、只是不打了）。進入第二階段。
3. **若有多隊並列最高（平手）** → 這些平手隊**各折損 500 兵**（不足 500 就扣光、損失不超過其投入），然後**喪失攻擊資格**，剩餘兵力退回原本的出發地。**比它們低的攻方仍留在場上**。
4. 重複 1~3（往下一層比），直到出現**唯一最高的攻方（突圍者）**，或所有攻方都退出。

> 重點：平手的最高兩（多）方會互相抵銷退場，讓**原本較低的攻方有機會突圍**。

### 第二階段：突圍者 vs 島上守軍（同 2.0）
- **突圍者 > 守軍** → **攻下該島**，扣掉守軍兵力後留駐；**守軍全滅**。
- **突圍者 < 守軍** → 守方**守住**，扣掉突圍者兵力後留駐；**突圍者全滅**。
- **突圍者 = 守軍** → **同歸於盡**，成無人空島。
- **島上無守軍（0 兵）** → 突圍者**直接佔領、不扣兵**。
- **沒有突圍者**（攻方全數抵銷退場）→ 守方**原封守住**；若原本就無守軍則維持空島。

### 兵力去向總整理
- **未突圍 / 平手退場**的攻方：兵力**退回原本的出發地**（島或兵營）；平手隊另**折損 500**（一次）。
- **戰敗的守方**：駐軍**全部消失**。
- **突圍成功者**：扣掉守軍兵力後的剩餘，留在該島當新駐軍。

### 完整範例
A、B、C、D、E 五隊都攻擊 F 的島；F 島上守軍 300：

| 小隊 | A | B | C | D | E | F（守） |
| :--: | :--: | :--: | :--: | :--: | :--: | :--: |
| 兵力 | 2000 | 2000 | 500 | 500 | 400 | 300 |

1. 最高 = 2000（A、B 平手）→ A、B **各折損 500**（→各 1500），退回出發地、出局。
2. 剩下最高 = 500（C、D 平手）→ C、D **各折損 500**（→各 0，扣光），出局。
3. 剩下 **E（400）為唯一最高 → 突圍者**。
4. E(400) vs 守軍 F(300)：400 > 300 → **E 攻下該島**，扣掉守軍 300 後**留 100 兵**；F 的 300 守軍全滅。

> 結果：該島由 **E 佔領、島上 100 兵**；A、B 各退回 1500、C、D 退回 0、F 守軍歸零。

## 六、資源發放與計分（含豐收）

攻擊結算完（且訓練完）後，各島依**最終佔領者**發放椰子：

- **大島**：每回合基礎 **800** 椰子。
- **小島**：每回合基礎 **500** 椰子。
- **豐收加成**（見下）。
- 椰子全歸**該島佔領者**（無人空島不產出；每島單一佔領者，不共享）。

### 豐收 🌾（保留）
- **每回合開始隨機指定 2 座島為豐收島**，該回合產出 **×1.5**。
- **最後一回合：全島大豐收**，所有島 **×1.5**。
- 豐收島會在地圖標示並於畫面頂端公告，讓各隊下指令前就能針對搶佔。

## 七、勝負

- 所有回合結束後，**椰子總數最多的小隊獲勝**，依椰子由多到少排名。

---

## 八、其他判定與防呆

- **兵力超支**：同隊同階段、自同一出發地移出的兵力總和超過該地現有 → 涉及該出發地的指令全部作廢（兵力留原地）。
- **攻擊自己的島**：無效。
- **移動到非己方領地**：無效。
- **移動每指令上限 500**；超過視為無效或截到 500（實作時提示重下）。
- **兵營絕對安全**：不可被攻擊。
- **守軍下限**：己方島嶼結算後守軍不足 100 → 強制中立（見第四節）。

## 九、與 2.0 的差異速覽

| 項目 | 2.0 | 3.0 |
| :--- | :--- | :--- |
| 玩家指令 | 開墾 / 移動 / 攻擊 | **移動 / 攻擊**（訓練改自動） |
| 島嶼 | 多島 | **8 小島 + 中央 1 中立大島（黃金洞窟），共 9 座** |
| 移動 | 無單指令上限 | **每指令最多 500 兵** |
| 養成 | 開墾（兵→島產出，1:1） | **自動訓練（每回合自動，兵營每滿1000 +200）** |
| 守軍下限 | 無 | **己島守軍不足 100 → 強制中立** |
| 攻擊平手 | 各損一半、一半撤回 | **各折損 500（一次）、其餘退回原地** |
| 退兵去向 | 一律回兵營 | **退回原本的出發地** |
| 結算順序 | 開墾→移動→攻擊→資源 | **移動→攻擊→自動訓練→資源** |
| 守軍裁決 | 突圍者 vs 守軍相減 | **同 2.0** |
| 豐收 / 產出 | 2島×1.5、末回合全島；大800/小500 | **同 2.0** |
