# 🎤 COCONUT WARS 3.0 · 主持人使用說明

3.0 的結算改成**「主持人步進模式」**：不再自動快速播完，而是**由你（主持人）按一下才揭曉下一步**。
你負責炒熱氣氛、賣關子、宣布結果；系統負責把每一步用大字卡清楚呈現。

---

## 一、開場準備

1. 一台電腦開 **主控台**：`http://<網址>/3.0/`（例如 `http://localhost:8000/3.0/`）。
2. 主控台右上角按 **「🖥️ 開啟播放視窗」**，把跳出來的**播放視窗拖到投影幕、按 F11 全螢幕**給全場看。
   - 播放視窗只會顯示畫面、**不會有任何按鈕**，玩家不會誤觸。
   - 主控台留在你自己的筆電，用來輸入指令、按結算、控制節奏。
3. 兩個視窗要在**同一台電腦、同一個瀏覽器**（它們會自動即時同步）。

> 建議：主控台筆電接投影幕（延伸桌面），播放視窗丟到投影幕全螢幕，你看筆電操作。

---

## 二、每回合流程

1. 各小隊把這回合的指令講給你 → 你在主控台**選隊伍、加指令**（移動 / 攻擊 / 訓練，每隊最多 3 個）。
2. 全部輸入完 → 按 **「結算本回合 ▸」**。
3. 進入**步進揭曉**：畫面中央會跳出大字卡，右下角有提示：
   > **按「空白鍵」或點「下一步 ▸」揭曉**（也可按 Enter 或 → 方向鍵）
4. 你**每按一下、揭曉一格**，一邊揭曉一邊講解。播放視窗會**同步**顯示同一格。
5. 最後一格是「本回合回顧」，按「完成 ✓」就進到下一回合。

---

## 三、揭曉會依序出現這些卡片

| 卡片 | 內容 | 你可以怎麼帶 |
| :--- | :--- | :--- |
| **第 X 回合 · 開戰！** | 本回合豐收島是哪幾座 | 「這回合金錢島會 ×1.5，大家盯緊了！」 |
| 🚶 **移動階段** | 有哪些調兵（小人在地圖上移動） | 快速帶過即可 |
| ⚔️ **○○島 爭奪戰** | 這座島**有誰來搶、各出多少兵、守軍多少** | **這裡最關鍵！先唸出參戰兵力、賣個關子再按下一步** |
| ⚔️ 爭奪戰**逐步結果** | 一步步揭曉：誰跟誰平手互扣、誰突圍、最後誰佔領 | 每按一步喊一句，最後一步島會閃光、宣布佔領者 |
| 🏋️ **訓練階段** | 哪些隊練兵 +多少 | 「X 隊悄悄練了 1000 兵…」 |
| 🥥 **資源結算** | 各島產出椰子（椰子在島上冒出來） | 帶到分數 |
| 📊 **本回合回顧** | 各隊**搶到/丟掉哪些島**、**本回合賺多少椰子**、目前**總兵力**（依椰子排序） | 宣布排名、點名亮點：「這回合最大贏家是…！」 |

> **只有「有爭奪的島」才會跳對戰卡**；沒人搶的島不佔時間。所以你可以把節奏集中在幾場關鍵大戰上。

## 四、主持小訣竅

- **賣關子**：在「爭奪戰」卡（顯示各隊出兵）停久一點，讓大家猜誰贏，再按下一步揭曉。
- **善用平手戲劇性**：3.0 平手會各折損 500 退場，常出現「兩強互扣、老三撿尾刀」，這是全場最嗨的點，用力渲染。
- **回顧卡點名**：唸出「＋島（綠色）＝搶到、－島（紅色）＝丟掉」，再宣布椰子排行，落後的隊也會被激起鬥志。
- **最後一回合**：卡片會顯示「全島大豐收 ×1.5」，提醒大家「這回合搶到就是雙倍分數，全力一搏！」

---

## 五、常見問題

- **按太快跳過了？** 沒關係，資訊都在，繼續往下帶；每格都有 `目前 X / 總數` 進度可參考。
- **播放視窗沒跟上？** 確認它和主控台是同一瀏覽器開的；重新整理播放視窗、主控台再按一次結算即可重新同步。
- **想加快？** 沒有爭奪的回合本來就很短；有大戰的回合就是要慢慢帶，那正是精彩處。
- **鍵盤**：`空白鍵` / `Enter` / `→` 都可以進到下一步。
