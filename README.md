# 🥥 COCONUT WARS 椰子戰爭 · 後台系統

給營隊使用的島嶼策略遊戲後台。共有兩個版本，**兩者完全獨立、互不影響**：

- **`1.0/`｜完整版**：金錢島 / 軍火島資源點、協防（help/accept）、結盟合法集合 + 四階衝突裁決、兵營利息、低兵力救濟。規則最完整、策略最深。
- **`2.0/`｜開墾版**：為「第一天剛認識的高中生」簡化——**移除資源點與跨隊協防/結盟的精確協調**，指令只剩 **開墾 / 移動 / 攻擊**，並加入豐收機制增添變化與翻盤性。

## 快速開始

因使用 ES Modules，需透過本機伺服器開啟：

```bash
cd cscampearthgame
python3 -m http.server 8000
# 開啟 http://localhost:8000        → 版本選擇頁
# 或直接 http://localhost:8000/2.0/ → 2.0 開墾版
```

兩版都支援**主控台 + 播放視窗**雙視窗（右上「🖥️ 開啟播放視窗」，同一瀏覽器內以 BroadcastChannel 同步）。

## 2.0 開墾版規則重點

- **指令（每隊每回合最多 3 個）**：
  - `開墾(己方島, 兵力)`：用**該島上的駐軍**投入（不需出發地），投入的兵**永久消耗**，該島每回合椰子產出 **+等量（1:1）**，加成隨島留給新主人。
  - `移動(出發, 目的, 兵力)`：在自己的兵營／島之間調度。
  - `攻擊(出發, 目標, 兵力)`：進攻他人島嶼，**無最低門檻**（100 的倍數即可）。
- **結算順序**：開墾 → 移動 → 攻擊 →（資源發放）。三階段依序執行。
- **攻擊裁決**：島上各勢力（各進攻方 + 守方）依兵力比高低——
  1. 找出目前**最高**兵力；若有**多方同為最高** → 這些方**各損一半後喪失攻擊資格**（存活的一半撤回兵營），再往下一層比。
  2. 反覆直到出現**唯一最高者** → 勝出，**扣掉次強者的兵力**後，剩餘兵力留駐該島；其餘落敗者（含守方）**全部消失**。
  - 例：A、B 各 2000，C、D 各 500，E 400，守方 F 300 →　A、B 各損 1000 出局 → C、D 各損 250 出局 → **E 以 400 擊敗 F(300)，佔領並留 100 兵**，F 全滅。
- **豐收**：每回合隨機 **2 座島 ×1.5** 椰子；**最後一回合全島大豐收 ×1.5**。

> 註：2.0 目前**未包含**兵營利息（×1.5）與低兵力救濟。若希望弱隊不被清空出局，可再加「低兵力救濟」，跟我說即可。
