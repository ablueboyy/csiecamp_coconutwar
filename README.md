# 🥥 COCONUT WARS 椰子戰爭 · 後台系統

給營隊使用的島嶼策略遊戲後台。目前的最終版本為 **`3.0/`｜訓練版**。

- **指令**：移動（每指令 ≤500 兵，第一回合不限）/ 攻擊 / 訓練。
- **訓練**：兵營每滿 1000 兵 +200 兵（每回合最多一次）。
- **攻擊平手**：各折損 500，其餘兵力退回原本的出發地。
- **結算順序**：移動 → 攻擊 → 訓練 → 資源發放（椰子）。
- **豐收**：每回合隨機 2 座島 ×1.5 椰子；最後一回合全島大豐收 ×1.5。

完整規則見 `3.0/docs/`：
- `玩家規則.md`｜給小隊員的規則
- `rules.md`｜完整規則書（含裁決細節與可調參數）
- `主持人使用說明.md`｜主控台操作說明

## 快速開始

因使用 ES Modules，需透過本機伺服器開啟：

```bash
python3 -m http.server 8000
# 開啟 http://localhost:8000        → 自動跳轉至 3.0
# 或直接 http://localhost:8000/3.0/ → 3.0 訓練版
```

支援**主控台 + 播放視窗**雙視窗（右上「🖥️ 開啟播放視窗」，同一瀏覽器內以 BroadcastChannel 同步）。
